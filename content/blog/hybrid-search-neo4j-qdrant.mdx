## Motivation

Vector search is great for semantic similarity, while graph search is great for relationship-aware traversal. Hybrid retrieval combines both.

## Code sample

```python
def hybrid_retrieve(query: str):
    vector_hits = qdrant.search(query)
    graph_hits = neo4j.traverse(vector_hits)
    return rerank(query, vector_hits + graph_hits)
```
